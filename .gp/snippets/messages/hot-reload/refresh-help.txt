Refreshes the preview browser for hot-reload on port 3005.
The hot-reload 'setup' and 'start' command must be run before this command will succeed. 
This command can only be used when the hot-reload server is running. The hot-reload server will be running 
whenever a successful hot-reload `start` command is given or whenever Laravel Mix Watch is started 
(and running) via yarn.

Usage:
  hot-reload refresh [flag]

Flags:
  -h, --help       help for help
